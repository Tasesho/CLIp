# CLIp

This project was made to learn the basics of Rust Lang, the goal of the project is, Make a CLI (Command Line Interface) assitant, It is inspired by another Project called 'clip_assistant' made by myself(link below). I will start another Repository to keep a **Clean** history of commits. Due to this project is a Rewritte of an older one **Anyone** can help in the development of this tool, For more information please visit: [CONTRIBUTING.md](./CONTRIBUTING.md)

## Main features!

The User can Interact with the Command Prompt. Following the next commands:

##### Hello
```CLIp salutes the User with a friendly message.```

##### Help
```Deploys a list with all the avaliable commands CLIp can do.```

##### GiveAdvice
```CLIp gives you an advice (in later updates there will be more)```

##### ShowHour
```show the actual date time.```


## Why CLip was developed?

CLIp was born to help the user in the terminal, the user can interact with CLIp so learning how to use and familiriaziting with the Terminal.
